# 燃やす埋める
## 基本
- 各問題を縦に並べる
- Node(s) → e選択枝1 → Node(問題) → e選択枝2 → Node(t) というグラフを作る
- 最大化する場合は、考えうる最大の利益をあらかじめ得てることにし、最小化問題に置き換える
- 制約条件を入れる時は、ある選択肢の組み合わせを選んだ時に、s-tカットが成立しないように辺を張る

- 一般的な場合
   ┌───→ A ───→┐
S ─┤           ├─ T
   └───→ B ───→┘

- A = S, B = T の時罰則Z
   ┌───→ A ───→┐
S ─┤     ↑Z    ├─ T
   └───→ B ───→┘

- A = S, B = S の時報酬Z
   ┌───→ A ───→┐
S ─┤     ↑Z    ├─ T
   └───→ B ───→┘
