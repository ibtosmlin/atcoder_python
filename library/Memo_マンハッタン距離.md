# マンハッタン距離

$d((x_0,y_0),(x_1,y_1)) = |x_0-x_1|+|y_0-y_1|$ という距離
$x,y: \to (u, v)=(x+y,x-y)$ に45度回転させると、
$d((u_0,v_0),(u_1,v_1)) = max(|u_0-u_1|, |v_0-v_1|)$となり
独立して考慮することができる
